### <img src="https://fonts.gstatic.com/s/e/notoemoji/latest/1f579/512.gif" width="28"> &nbsp; How to Play

</div>

<br/>

| Step | Action | Details |
|:----:|--------|---------|
| `01` | **Select** | Click a tube — the top alien lifts up |
| `02` | **Move** | Click another tube — alien moves there |
| `03` | **Rule** | Only valid if colors match or tube is empty |
| `04` | **Win** | All tubes filled with one color = Level Complete |
